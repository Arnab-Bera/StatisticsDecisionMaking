# Statistics and Decision Making Assignment

This repository contains the assignment work for Statistics and Decision Making, including data analysis and decision-making models.

## Files

- `Arnab_Bera_PPDSSA_B2.ipynb`: Jupyter notebook with analysis and visualizations.
- `StatisticalAnalysis_DecisionMaking_Assignment.ipynb`: Main assignment notebook containing statistical analysis.
- `property.csv`: Dataset file used for the analysis.

## Requirements

- Python 3.x
- Jupyter Notebook
- Libraries: pandas, numpy, matplotlib, seaborn (install via `pip install pandas numpy matplotlib seaborn`)

## Usage

1. Clone the repository.
2. Open the `.ipynb` files in Jupyter Notebook or VS Code.
3. Run the cells to execute the analysis.
